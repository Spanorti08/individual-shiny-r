# individual-shiny-r

This repository contains a simple Shiny dashboard built in **R** for the DSCI 532 individual assignment.

## Requirements

Install required packages in R:

install.packages(c("shiny", "tidyverse"))

## Run the app

Run the following in R:

library(shiny)
runApp()
