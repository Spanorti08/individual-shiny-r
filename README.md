# individual-shiny-r

This repository contains a simple **Shiny dashboard built in R** for the **DSCI 532 Individual Assignment**.

## Requirements

Install the required R packages:

install.packages(c("shiny", "ggplot2", "dplyr"))

## Run the app locally

Run the following in R:

library(shiny)
runApp()
